# Site images — replace these with your own

All placeholders were pulled from the internet. To use your own photos, just
overwrite the files below **keeping the same filenames** — no code changes needed.

| File | Used for | Recommended size |
|---|---|---|
| `hero/car-1.jpg` … `car-4.jpg` | Home hero slideshow backgrounds | 1600×900, landscape |
| `tiles/suv.jpg` | "SUVs & 4x4s" tile | 800×600 |
| `tiles/pickup.jpg` | "Pickups & Trucks" tile | 800×600 |
| `tiles/motorcycle.jpg` | "Motorcycles" tile | 800×600 |
| `tiles/garage.jpg` | "Service Parts" tile | 800×600 |
| `tiles/laptop.jpg` | Laptops promo card | 800×600 |
| `tiles/phone.jpg` | Mobile Phones promo card | 800×600 |

Headline text for each hero slide lives in `components/HeroCarousel.tsx` (SLIDES).
Keep hero images under ~300KB each — they load on every visit to the home page.
